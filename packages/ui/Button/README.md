# button

## Installation

```sh
$ yarn add @bits-x/button
# or
$ npm install @bits-x/button
```

## Usage
```js
import Button from '@bits-x/button'

<Button>
  Primary Button
</Button>
<Button disabled>
  Primary Button
</Button>
<Button size="small">
  Primary Button
</Button>
<Button variant="outlined">
  Secondary Button
</Button>
<Button variant="error">
  Delete Button
</Button>
<Button type="submit">
  Guardar
</Button>
```
