# button

## Installation

```sh
$ yarn add @cometa/button
# or
$ npm install @cometa/button
```

## Usage
```js
import Button from '@cometa/button'

<Button>
  Primary Button
</Button>
<Button disabled>
  Primary Button
</Button>
<Button size="small">
  Primary Button
</Button>
<Button variant="outlined">
  Secondary Button
</Button>
<Button variant="error">
  Delete Button
</Button>
<Button type="submit">
  Guardar
</Button>
```
